Dailies Command Center — Help
Reference guide for reviewers and managers
Workspaces & Views
Detail View
The standard QA view. Focuses on one project at a time, providing full schedule, velocity, and AI diagnostic breakdowns.
Grid View
High-density overview. Displays multiple projects at once. Use the Pivot Selector in the dock to switch between Schedule, Velocity, Notes, or Deliverables data.
Deck Mode
Presentation mode. Formats projects into 16:9 slides. Ideal for morning meetings. Allows you to "Stage" multiple projects for batch email exports.
Theater Mode
Available in Deck mode. Blacks out the surrounding UI for distraction-free presentations. Use Arrow Keys or Spacebar to navigate slides.
Daily Workflow
1
Sync QB — Pull the latest project data from QuickBase. Do this when QB statuses or BOMs have changed.
2
Run Review — Prompts for a date, processes daily vendor files, and loads anomalies into the queue.
3
Admin Checks — Open the right-side Review Hub to clear Status Mismatches, verify Special Crossings, and log missing BOMs.
4
Review & Note — Check the Vendor Update and System Flags. The AI (Gemini) pre-drafts a manager note. Edit it to reflect your decision.
5
Commit — Saves the review and moves the project to the Reviewed drawer.
6
Export — In the Reviewed drawer (or via Deck Staging), export the daily summaries to CSV or Email.
Dynamic Filter Dock
The floating dock at the top of the screen adapts to your current workspace.
Search
Instantly filter the queue by FDH ID, City, or Vendor.
Filters
Narrow projects by Health (Critical/Warning), Vendor, City, or Target OFS month.
Stage Project
(Deck Mode) Adds the current project to a batch queue. Allows you to email a combined PDF/text report for all staged projects at once.
Network Path & Dependencies
The horizontal strip in the Detail View showing physical fiber dependencies.
Green Dot
Node has light to cabinet confirmed (or is OFS/Complete).
Amber Dot
Node is currently waiting on light.
⚡ Transport
Transport (backhaul) is ready, meaning this project bypasses upstream physical dependencies for light.
Click to Walk
Click any upstream or downstream node to immediately jump to that project's dashboard.
Field Velocity & Production
Pace Tracker
Calculates the daily footage/NAPs required to hit the Target Date. Turns red if the required pace exceeds standard thresholds.
Days Left
Calendar days remaining until the target date. Overdue projects display in red.
+ TODAY
Green badges indicate fresh production was logged in the latest daily report.
Global Construction Timeline (Gantt)
Bar Color
Matches the project's current stage/status. Overdue projects display in red. Hover a bar to see a summary HUD.
Beads
Small markers on the bar: L = Locates called, C = Cabinets set, ⚡ = Light present, ★ = Phase 100% complete.
EOM Spike
A concentration of projects ending at or near end-of-month. Hover for a breakdown of which vendors and BSL counts are involved.
Flag Glossary
Status Mismatch
Project's QB status does not match the field report. Update QB.
Check Crossings
Special crossings detected. Confirm permit status, bore depths, and restoration plan with the vendor.
Please Input BOM
Main BOM quantities are zero. Add BOMs to QuickBase so the engine can calculate velocity.
Blocked by Upstream
The project feeding this FDH is not yet lit, and Transport is not ready to override it.
Delaying Downstream
This project is not lit, and is actively holding up a downstream project that lacks transport.
Transport Override
Transport is available, bypassing the physical upstream dependency.
Overrun
Vendor has reported total footage exceeding the QuickBase BOM by more than 5%.
Possible Reroute
Vendor reported activity on a phase (like UG or Strand) where the BOM is 0. Indicates a possible field design change.
BOM Discrepancy
The vendor's reported BOM total does not match QuickBase's BOM total.
Ghost Report
A project is marked active but the vendor did not submit a daily report today.
CD Intel
Gemini detected major crossing risks (Highways, Railroads, Rivers) in the Construction Drawing PDF.
