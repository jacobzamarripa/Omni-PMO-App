Omni PMO — Intelligence Guide
Comprehensive reference for review orchestration
Core Operating Model
The Omni PMO App is a high-performance orchestration layer designed to bridge field data from vendors with PM project management. It utilizes the GlassFlow Shell for a context-aware mobile experience.
Workspaces & Layouts
Diagnostics Queue
Your primary inbox. Projects with anomalies appear here. Switch between Inbox, List, and Grid views using the toggle in the toolbar.
Detail Surface
Deep-dive into a single project. Access Schedule Progress, Field Velocity, and the Field Thread. Use the dock to commit reviews or skip to the next project.
Gantt Interactive
Rotate your device to landscape to trigger the full-screen Gantt view. This isolated workspace provides high-density timeline visualization and focus mode.
System Actions
Sync QuickBase: Pulls the latest reference data from the QB API. Critical for keeping the local dictionary in sync with system-of-record changes.
Run Daily Review: Executes the diagnostic engine (Benny Engine) to process raw vendor reports and identify new anomalies.
↻
Refresh Interface: Performs a fast re-read of the mirror sheets to update the UI without re-running diagnostic rules.
Review Hub (Admin)
Accessible via the round icon in the Queue. Manages cross-cutting tasks:
Crossings
Permit or construction verification needed for special project intersections.
Status Mismatch
Auto-detected discrepancies between QB status and field reporting.
BOM Checks
Zero-quantity BOM validation to ensure accurate velocity calculation.
Field Thread & AI
Use AI Draft in the detail view to generate concise project summaries based on vendor comments and diagnostic data. All notes are saved to the backend for audit trail.
Shortcuts & Gestures
Swipe Right
Swipe this help panel right to dismiss it instantly.
Tap Badge
Tap badges in the Queue to jump to specific diagnostic sections.
